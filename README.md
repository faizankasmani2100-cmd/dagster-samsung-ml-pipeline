# Dagster Samsung ML Pipeline

This project demonstrates a reproducible machine learning pipeline using Dagster
implemented in Google Colab.

## Dataset
Samsung Electronics stock data (005930.KS) sourced via Yahoo Finance.

## Pipeline Steps
- Data ingestion
- Data cleaning
- Exploratory data analysis
- Feature engineering
- Model training
- Evaluation

## Key Advantage
Dagster enables selective re-execution of pipeline components,
saving significant time during experimentation.

## Output
- Dagster asset dependency graph
